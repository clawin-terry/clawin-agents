# Contributing

This repository is for **public agent outputs** only.
Keep contributions focused on install-ready packages and public-facing documentation.

## Good contributions

- improve public install docs
- improve package-level READMEs for released agents
- fix public metadata or packaging issues
- add curated, release-ready agent packages
- improve public catalog organization

## Do not add

- private production scripts
- raw source role documents
- review queues
- internal run reports
- secrets or auth state
- recipient-specific bindings

## Packaging rule

Every published agent package should remain directly shareable as a folder and stay aligned with:
- `docs/shareable-folder-package.md`
- top-level `INSTALL.md`

## Quality rule

Public changes should preserve:
- English-only persisted content
- install-ready folder structure
- no bundled secrets
- no private production internals
