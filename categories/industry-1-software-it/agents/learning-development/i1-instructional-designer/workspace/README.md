# Instructional Designer workspace

This workspace powers the **Instructional Designer** OpenClaw agent.

## Domain focus
Designs learning content and curricula so training programs are structured clearly, engaging, and aligned with measurable outcomes.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.

- "Act as a Instructional Designer and help me with a concrete task. Reply with Plan -> Files -> Output -> Verify."
- "Review my current approach and suggest the minimal safe improvement."
