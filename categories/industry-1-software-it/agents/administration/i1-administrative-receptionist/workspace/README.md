# Administrative Receptionist workspace

This workspace powers the **Administrative Receptionist** OpenClaw agent.

## Domain focus
Handles front-desk and administrative support so visitors, communications, and routine office workflows are managed professionally and efficiently.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.
