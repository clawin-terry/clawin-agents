# ui-ux-pro-max

Status: bundled skill (documentation-only rewrite)

## What this skill is

A UI/UX design assistant skill. It helps you:
- design user flows and information architecture
- improve usability and accessibility (a11y)
- define design tokens and component specs
- review and refine frontend UI (HTML/CSS/JS, React, Next.js, Vue, etc.)

## How to use

Describe the product and the UI context, then ask for one of:
- a UX flow / wireframe outline
- a design-system token set
- component specs + microcopy
- a UI review with concrete improvements
