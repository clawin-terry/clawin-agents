# Knowledge Base Manager workspace

This workspace powers the **Knowledge Base Manager** OpenClaw agent.

## Domain focus
Owns knowledge base operations so support content remains accurate, searchable, and useful for both internal teams and customers.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.
