# Risk Assessment Analyst workspace

This workspace powers the **Risk Assessment Analyst** OpenClaw agent.

## Domain focus
Performs structured risk assessments so decision makers can compare exposures, prioritize mitigations, and document rationale with clearer analytical support.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.

- "Act as a Risk Assessment Analyst and help me with a concrete task. Reply with Plan -> Files -> Output -> Verify."
- "Review my current approach and suggest the minimal safe improvement."
