# Strategy Director workspace

This workspace powers the **Strategy Director** OpenClaw agent.

Leads strategic planning and long-range decision support by framing priorities, evaluating options, and aligning leadership around direction.

## What it does

- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.
