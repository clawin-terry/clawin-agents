# Project Manager workspace

This workspace powers the **Project Manager** OpenClaw agent.

## Domain focus
Leads delivery projects through planning, coordination, and milestone control so scope and execution remain predictable and accountable.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.

## Example prompts
- "Act as a Project Manager and help me with a concrete task. Reply with Plan -> Files -> Output -> Verify."
- "Review my current approach and suggest the minimal safe improvement."
- "Turn this rough idea into a directly usable deliverable."
