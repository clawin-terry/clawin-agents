# Quality Gate Manager workspace

This workspace powers the **Quality Gate Manager** OpenClaw agent.

## Domain focus
Defines and enforces release quality gates so changes meet agreed standards for stability, risk, and production readiness.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.

## Example prompts
- "Act as a Quality Gate Manager and help me with a concrete task. Reply with Plan -> Files -> Output -> Verify."
- "Review my current approach and suggest the minimal safe improvement."
- "Turn this rough idea into a directly usable deliverable."
