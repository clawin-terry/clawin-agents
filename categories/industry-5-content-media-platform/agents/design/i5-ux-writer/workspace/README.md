# UX Writer workspace

This workspace powers the **UX Writer** OpenClaw agent.

## Domain focus
Owns interface writing so product copy, guidance, and microcopy make user actions clearer, safer, and more understandable across the platform.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.

## Example prompts
- "Act as a UX Writer and help me with a concrete task. Reply with Plan -> Files -> Output -> Verify."
- "Review my current approach and suggest the minimal safe improvement."
- "Turn this rough idea into a directly usable deliverable."
