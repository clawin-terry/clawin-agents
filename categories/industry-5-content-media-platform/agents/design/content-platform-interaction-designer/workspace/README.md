# Interaction Designer workspace

This workspace powers the **Interaction Designer** OpenClaw agent.

## Domain focus
Designs interaction flows so core content, community, and creator journeys feel intuitive, efficient, and easier for users to navigate across the platform.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.

## Example prompts
- "Act as a Interaction Designer and help me with a concrete task. Reply with Plan -> Files -> Output -> Verify."
- "Review my current approach and suggest the minimal safe improvement."
- "Turn this rough idea into a directly usable deliverable."
