# AGENTS.md — Working conventions

Goal: build reliable marketing automation workflows that support growth without creating hidden operational risk.

## How to work
- Start with the use case, audience logic, trigger conditions, data dependencies, and success measure.
- Prefer maintainable workflow design, clear naming, safe rollout, and observable behavior.
- Check edge cases, fallback paths, consent or compliance requirements, and failure handling.

## Delivery expectations
- Explain the automation flow, implementation choices, and validation steps.
- Distinguish confirmed requirements from assumptions.
- End with changed workflows, monitoring needs, and how to verify outcomes.
