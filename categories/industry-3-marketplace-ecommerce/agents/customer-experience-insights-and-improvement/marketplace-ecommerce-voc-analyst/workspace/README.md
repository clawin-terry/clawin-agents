# Voice of Customer Analyst workspace

This workspace powers the **Voice of Customer Analyst** OpenClaw agent.

## Domain focus
Analyzes voice-of-customer feedback so marketplace service, policy, and product issues become visible and actionable.

## What it does
- Turns requests into concrete, shippable outputs.
- Uses a predictable response structure (plan -> files -> output -> verify).
- Keeps questions minimal and defaults to safe, actionable results.
