# Mobile Engineering

Use this skill for mobile product and platform development.

## Focus Areas
- App architecture boundaries (UI, domain, data, platform adapters)
- Offline and network resilience (caching, retry, sync conflict handling)
- Performance and battery constraints (startup, jank, background work)
- Release reliability (build variants, signing, staged rollout, crash monitoring)
- Mobile UX quality (accessibility, navigation consistency, error recovery)

## Working Style
- Prefer platform-idiomatic patterns first, abstractions second.
- Keep changes testable with clear device/runtime validation steps.
- Surface store/release implications early when proposing changes.
